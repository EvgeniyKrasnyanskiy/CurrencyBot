# 💵 CurrencyBot (USDT/RUB)

Telegram-бот для автоматического мониторинга курса USDT к рублю через API **Rapira** и **CoinGecko**. Предназначен для крипто-сообществ и майнеров.

## ✨ Основные функции
* **Умный закреп:** Автоматически обновляемое сообщение в канале с текущим курсом.
* **Индикация тренда:** Визуальные стрелки изменения курса (`⬈`, `⬊`, `→`) относительно предыдущего замера.
* **Дневная статистика:** Расчет изменения курса в процентах с начала текущих суток (с 00:00).
* **Система алертов:** Уведомления о резких скачках волатильности (от 2% до 6% за периоды от 1ч до 24ч).
* **Кэширование:** Оптимизация запросов к API для обхода лимитов.

## 🛠 Технологии
* **Python 3.x**
* **python-telegram-bot** (v13.x)
* **Requests** — работа с API и прокси.
* **Dotenv** — безопасное хранение конфигурации.

## 🚀 Быстрый запуск

1. **Установка зависимостей:**
   ```bash
   pip install -r requirements.txt
Настройка окружения:
Создайте файл .env на основе .env.example и заполните:

BOT_TOKEN — токен от @BotFather.

CHANNEL_ID — ID канала или группы.

ADMINS — список ID администраторов через запятую.

SLEEP_TIME — интервал обновления (сек), например 600.

Запуск:

Windows: запустите start_bot.ps1 или:

Bash
python currency_bot.py
📈 Логика отображения
Бот формирует сообщение в формате:

💵 USDt = ₽93.10 ⬈ | ₽92.85 → [22:05]

📈 16.04: +1.45% (Rap) | -0.12% (CG)

🛡 Безопасность
Файл конфигурации .env добавлен в .gitignore.

Реализован файл блокировки currency_bot.lock для предотвращения запуска нескольких копий бота.
